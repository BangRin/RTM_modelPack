# ReadMe

## 도움을 주신분들
- 메이테츠600v씨의 Minato Commuter Train_A 개조품입니다.
- 森林研究所씨의 부품이 포함되어있습니다.
- Kamikawa씨의 ATS 3D모델이 포함되어있습니다.

- presscenter 오류해결 도움
- Gurrmy_ 광고, 버튼, LCD, 도색, 디자인 가이드 제공
